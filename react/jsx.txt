- JSX is a declarative syntax to describe what components look like
- Components must return a block of JSX
- JSX (JavaScript XML) is a syntax extension of JavaScript that lets you write HTML-like code inside JavaScript.
- Browser does not understand JSX so, it convert JSX into javascript using a tool Babel (automatically included while creating project from create react app),
ex- 
<header>
    <h1 style="color: red>
        Hello React !!
    </h1>
</header>

This code will be converted into something like - 

React.createElement(
    'header',
    null,
    React.createElement(
        'h1',
        { style: { color: 'red' } },
        'Hello React !!'
    )
);

- You can use React without JSX.


There are 2 options for setting react project
1. create-react-app

- This is a starter kit for building reaact applications.
- Major things are pre configured like ESlint, Prettier, Jest etc.
- It uses slow and outdated technologies (webpack) for bundling, hence not recommended to use for real world applications now. You still can use it for self learning projects

2. Vite
- This is a build tool that have template for setting the react application.
- You have to manually add ESlint configuration etc.
- Uses fast Hot Module Replacement (HMR) and bundling, hence, now recommended for real world applications

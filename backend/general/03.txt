TYPES OF API REQUEST 

1. GET REQUEST
This request is used to get or fetch the data from the server
This API request is only used for fetching the data

2. POST REQUEST
This request is used to create a resource on the server

3. PUT REQUEST
This request is used to update the existing resource on the server

4. DELETE REQUEST
This request is used to delete the request on the server


5. PATCH REQUEST
This request is used to update the resource on the server but partically (only the passed fields)
